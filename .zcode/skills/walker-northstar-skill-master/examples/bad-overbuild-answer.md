# Bad Overbuild Answer

Avoid answers like this:

```text
Let's build a complete user system with registration, email verification, password reset, profile
pages, follow relationships, feeds, idea marketplace, collaboration rooms, and moderation, because
NorthStar will need these later.
```

Why this is wrong:

- It confuses AdgaiWalker with NorthStar.
- It builds platform complexity before the personal loop is proven.
- It optimizes accounts instead of answer matching.
- It delays the first useful user path.
