# Feature Scope Example

Request:

```text
I want users to log in and tell me what they need, so the agent can match them better.
```

Correct scope:

```text
Stage: AdgaiWalker personal system

This is an invite-only identity understanding entrance.

In scope:
- invite code
- lightweight session
- five profile questions
- question submission
- demand event
- matching context
- privacy and deletion request

Out of scope:
- full registration
- password reset
- profile pages
- social graph
- feed
- NorthStar accounts
```
