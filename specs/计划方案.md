## 0. 已完成清单（不再作为剩余开发项）

以下能力已在当前代码中存在，后续只做验收或缺陷修复：

- **Identity**：注册、登录、me、GitHub OAuth、密码重置、邮箱验证、邀请码、申请制、compass profile 更新。
- **Capability**：`GET /api/platform/capabilities?site=campus|compass` 两站能力查询。
- **AI Gateway**：统一 `/api/ai-gateway/chat`，前端不直连智谱；AI key 只走后端环境变量。
- **Campus 闭环**：空间、文章、帖子、回复、有帮助、有变化、搜索、Feed、收藏、AI 写作、通知、法律页、数据导出、注销、空间认领入口。
- **Compass 闭环**：工具/专题/文章/资讯列表与详情页、内容 CRUD + 版本、方案生成/保存/导出/反馈、搜索、收藏、用户中心、通知中心、资料编辑、注销入口。
- **Moderation**：任务 CRUD、状态流转、审核副作用、pending 任务去重。
- **Notification**：站内通知 + dev provider + SMTP provider + 投递记录。
- **Analytics / Insights**：行为事件上报基础服务、指标聚合、`/api/insights/*`、后台 insights / AI 成本 / 内容质量标签页。
- **Billing**：额度、账本、手动支付。
- **Compliance**：法律文档、同意、数据导出、账号注销。
- **Admin**：总览、审核、用户、内容、详情、合规、配置、Feature Flag、数据、支付、投递、内容质量报告。
- **技术债清理**：校园站 mock 清除、旧 AI 路由删除、`search:refresh`、admin typecheck、90 天行为数据清理。

## 2. R1：字段与契约收口

### R1-1 校园搜索响应字段统一

**问题**：旧文档曾要求 `matchStatus: "exact" | "partial" | "none"`，当前 shared 契约为 `CampusSearchMode = 'exact' | 'partial' | 'empty'`。

**落点**：
- `NorthStar/packages/shared/src/campus-contract.ts`
- `NorthStar/packages/server/src/modules/campus/routes.ts`
- `NorthStar/packages/frontlife-web/src/services/api.ts`
- `specs/架构设计.md` 中校园搜索相关描述

**验收**：
- [ ] 文档、shared 契约、server 响应、frontlife-web 消费字段使用同一命名。
- [ ] 空结果统一为 `empty` 或统一为 `none`，不能混用。
- [ ] 搜索 exact / partial / empty 三态各有测试或手动验收记录。

## 4. R3：专项 smoke 测试

### R3-1 数据隔离 smoke

**验收**：
- [ ] cn 请求不能读到 com-only 行为数据。
- [ ] com 请求不能采集中国敏感个人信息字段。
- [ ] `accounts` 与账号级等级共享，但 campus/compass profile 分离。

### R3-2 AI fallback smoke

**验收**：
- [ ] 无 `AI_API_KEY` 时返回 `mode: "demo"` 和中文降级说明。
- [ ] 敏感词输入被拒绝。
- [ ] AI 输出命中敏感词时进入 fallback。
- [ ] 额度不足时返回明确错误或 demo 降级。

### R3-3 认证与 token smoke

**验收**：
- [ ] password reset 后旧 token 不可继续使用。
- [ ] 账号禁用后旧 token 不可继续使用。
- [ ] `LOCK_SITE=1` 时客户端 header 不能改写服务端站点。

### R3-4 空态与断 server smoke

**验收**：
- [ ] 前端在空列表、空通知、空搜索、空内容质量报告时有可读状态。
- [ ] server 断开时前端展示中文错误态，不白屏。

## 5. R4：运营级体验补强

此阶段不预设“缺失”，只根据 R2/R3 验收结果拆任务。常见补强项包括：

- 文案：AI 生成、demo fallback、额度不足、内容审核状态。
- 权限态：无权限创建内容、无权限审批、未登录访问用户中心。
- 空态：无通知、无收藏、无方案、无搜索结果、无行为数据。
- 审计：后台关键操作是否写入 audit log。
